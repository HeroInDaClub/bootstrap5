Введение

Начните работу с Bootstrap и часто используемыми компонентами: адаптивная шапка, сетка, карусель, галерея, формы, уведомления и служебные классы.

Быстрый старт

Для работы большинства компонентов нужен CSS Bootstrap и пакет JavaScript bootstrap.bundle.min.js. Его лучше подключать внизу страницы перед закрывающим тегом </body>.

CSS

Скопируйте и вставьте таблицу стилей в <head> перед вашим style.css.

<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.0.1/dist/css/bootstrap.min.css" rel="stylesheet">

JS

Для dropdown, collapse, carousel, toast и burger menu нужен JavaScript.

<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.0.1/dist/js/bootstrap.bundle.min.js"></script>

Navbar

Рабочая копия твоей шапки. Здесь исправлены частые ошибки: убран лишний href, убран пустой <li>, у dropdown есть aria-labelledby, а для Bootstrap 5 используется data-bs-toggle="dropdown".

<nav class="navbar navbar-expand-md navbar-light bg-white border-bottom mb-4">
  <div class="container">

    <a class="navbar-brand" href="index.html">
      <img src="img/logo.png" alt="Логотип" style="width: 120px;">
    </a>

    <button class="navbar-toggler"
            type="button"
            data-bs-toggle="collapse"
            data-bs-target="#mainMenu"
            aria-controls="mainMenu"
            aria-expanded="false"
            aria-label="Открыть меню">
      <span class="navbar-toggler-icon"></span>
    </button>

    <div class="collapse navbar-collapse" id="mainMenu">
      <ul class="navbar-nav mx-auto mb-2 mb-md-0">

        <li class="nav-item">
          <a href="index.html" class="nav-link">Главная</a>
        </li>

        <li class="nav-item">
          <a href="contacts.html" class="nav-link">Контакты</a>
        </li>

        <li class="nav-item">
          <a href="blog.html" class="nav-link">Блог</a>
        </li>

        <li class="nav-item">
          <a href="price-list.html" class="nav-link">Прайс-лист</a>
        </li>

        <li class="nav-item">
          <a href="form.html" class="nav-link active">Оставить заявку</a>
        </li>

        <li class="nav-item dropdown">
          <a href="#"
             class="nav-link dropdown-toggle"
             id="catalogDropdown"
             role="button"
             data-bs-toggle="dropdown"
             aria-expanded="false">
            Портфолио
          </a>

          <ul class="dropdown-menu" aria-labelledby="catalogDropdown">
            <li>
              <a href="port1.html" class="dropdown-item">
                портретная съемка
              </a>
            </li>

            <li>

    </div>

  </div>
</nav>

Обязательное подключение JS

Если dropdown или бургер не открывается, почти всегда не подключен bootstrap.bundle.min.js или подключен не той версии.

<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.0.1/dist/js/bootstrap.bundle.min.js"></script>

Dropdown

В Bootstrap 5 используется data-bs-toggle. Старый вариант data-toggle относится к Bootstrap 4.

<li class="nav-item dropdown">
  <a class="nav-link dropdown-toggle" href="#" role="button" data-bs-toggle="dropdown">
    Портфолио
  </a>

  <ul class="dropdown-menu">
    <li><a class="dropdown-item" href="port1.html">Портретная съемка</a></li>
    <li><a class="dropdown-item" href="port2.html">Свадебные съемки</a></li>
  </ul>
</li>

Сетка

Основная схема Bootstrap-сетки: container → row → col.

<div class="container">
  <div class="row g-3">
    <div class="col-12 col-md-6 col-lg-4">Блок</div>
    <div class="col-12 col-md-6 col-lg-4">Блок</div>
    <div class="col-12 col-md-6 col-lg-4">Блок</div>
  </div>
</div>

Порядок колонок

Классы order-* позволяют менять порядок блоков на разных экранах.

<div class="row align-items-center">
  <div class="col-12 col-md-6 order-2 order-md-1">
    Текст
  </div>

  <div class="col-12 col-md-6 order-1 order-md-2">
    Картинка
  </div>
</div>

Spacing

m — внешний отступ, p — внутренний. x — слева и справа, y — сверху и снизу.

mt-3       /* margin-top */
mb-4       /* margin-bottom */
mx-auto    /* margin-left/right auto */
px-3       /* padding-left/right */
py-5       /* padding-top/bottom */
mx-2 mx-md-5

Carousel

Полный код карусели на 3 фотографии. Если будешь расширять до 9 фото, добавляй новые button в индикаторы и новые carousel-item строго внутрь carousel-inner.

<div id="carouselExampleIndicators" class="carousel slide" data-bs-ride="carousel">

  <div class="carousel-indicators">
    <button type="button"
            data-bs-target="#carouselExampleIndicators"
            data-bs-slide-to="0"
            class="active"
            aria-current="true"
            aria-label="Slide 1"></button>

    <button type="button"
            data-bs-target="#carouselExampleIndicators"
            data-bs-slide-to="1"
            aria-label="Slide 2"></button>

    <button type="button"
            data-bs-target="#carouselExampleIndicators"
            data-bs-slide-to="2"
            aria-label="Slide 3"></button>
  </div>

  <div class="carousel-inner">
    <div class="carousel-item active">
      <img src="img/carousel/1.jpg" class="d-block w-100" alt="Фото 1">
    </div>

    <div class="carousel-item">
      <img src="img/carousel/2.jpg" class="d-block w-100" alt="Фото 2">
    </div>

    <div class="carousel-item">
      <img src="img/carousel/3.jpg" class="d-block w-100" alt="Фото 3">
    </div>
  </div>

  <button class="carousel-control-prev"
          type="button"
          data-bs-target="#carouselExampleIndicators"
          data-bs-slide="prev">
    <span class="carousel-control-prev-icon" aria-hidden="true"></span>
    <span class="visually-hidden">Previous</span>
  </button>

  <button class="carousel-control-next"
          type="button"
          data-bs-target="#carouselExampleIndicators"
          data-bs-slide="next">
    <span class="carousel-control-next-icon" aria-hidden="true"></span>
    <span class="visually-hidden">Next</span>
  </button>

</div>

CSS для одинаковой высоты изображений

.carousel img {
  height: 800px;
  object-fit: cover;
}

Изображения

img-fluid делает изображение адаптивным. Для фиксированной формы используйте aspect-ratio и object-fit: cover.

<img src="img/photo.jpg" class="img-fluid w-100 rounded shadow"
     style="aspect-ratio:16/9; object-fit:cover;">

Masonry

Для простой галереи можно использовать CSS columns. Для полноценной раскладки — Masonry JS.

.masonry {
  columns: 3;
  column-gap: 10px;
}

.masonry img {
  width: 100%;
  margin-bottom: 10px;
  break-inside: avoid;
}
<script src="https://cdn.jsdelivr.net/npm/masonry-layout@4.2.2/dist/masonry.pkgd.min.js"></script>

<script>
window.addEventListener('load', function () {
  const grid = document.querySelector('#photo');

  new Masonry(grid, {
    itemSelector: '.col-12',
    percentPosition: true
  });
});
</script>

Cards

Для карточек не задавайте фиксированную ширину width: 18rem, если карточка стоит внутри Bootstrap-сетки.

<div class="row g-3">
  <div class="col-12 col-sm-6 col-lg-4">
    <div class="card h-100 shadow-sm">
      <img src="img/photo.jpg" class="card-img-top">
      <div class="card-body d-flex flex-column">
        <h5 class="card-title">Заголовок</h5>
        <p class="card-text">Описание...</p>
        <a href="#" class="btn btn-dark w-100 mt-auto">Подробнее</a>
      </div>
    </div>
  </div>
</div>

Таблицы

Чтобы таблица имела скругление, заверните ее в блок с rounded и overflow-hidden.

<div class="shadow rounded overflow-hidden">
  <table class="table table-bordered mb-0">
    ...
  </table>
</div>

Формы

Ширину формы лучше задавать через колонку, а не через w-50 на row.

<div class="row justify-content-center">
  <div class="col-12 col-md-8 col-lg-6">
    <form>
      <label for="Phone" class="form-label mt-3">Номер телефона</label>
      <input type="tel" class="form-control" id="Phone">

      <label for="Name" class="form-label mt-3">Ваше имя</label>
      <input type="text" class="form-control" id="Name">

      <button type="button" class="btn btn-dark btn-lg w-100 mt-3" onclick="validateForm()">
        Отправить
      </button>
    </form>
  </div>
</div>

Маска телефона

pattern проверяет формат, но не расставляет пробелы. Для автоматического форматирования нужна библиотека.

<script src="https://unpkg.com/imask"></script>

<script>
IMask(document.getElementById('Phone'), {
  mask: '+{7} (000) 000-00-00'
});
</script>

Toasts

Полный рабочий вариант: HTML-разметка toast, кнопка и JavaScript-функция. Toast должен быть на странице до подключения script.js, а Bootstrap JS должен быть подключен раньше твоего скрипта.

HTML

<button type="button"
        class="btn btn-dark btn-lg mt-3 w-100"
        onclick="validateForm()">
  Отправить
</button>

<div class="toast-container position-fixed bottom-0 end-0 p-3" style="z-index: 9999">
  <div id="successToast"
       class="toast"
       role="alert"
       aria-live="assertive"
       aria-atomic="true"
       data-bs-delay="5000">

    <div class="toast-header">
      <strong class="me-auto">Заявка</strong>
      <small>сейчас</small>
      <button type="button"
              class="btn-close"
              data-bs-dismiss="toast"
              aria-label="Close"></button>
    </div>

    <div class="toast-body">
      Заявка успешно отправлена!
    </div>

  </div>
</div>

JS

function validateForm() {
  const name = document.getElementById("Name").value.trim();
  const phone = document.getElementById("Phone").value.trim();

  if (name !== "" && phone !== "") {
    const toastEl = document.getElementById("successToast");
    const toast = bootstrap.Toast.getOrCreateInstance(toastEl);

    toast.show();
  } else {
    alert("Пожалуйста, заполните все поля формы.");
  }
}

Подключение скриптов

<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.0.1/dist/js/bootstrap.bundle.min.js"></script>
<script src="script.js"></script>

Липкий подвал

Подвал должен быть прямым элементом body, а основная часть страницы должна растягиваться.

<body class="d-flex flex-column min-vh-100">

  <main class="flex-grow-1">
    Контент
  </main>

  <footer class="mt-auto">
    Подвал
  </footer>

</body>

Полезные мелочи

Цвет active/focus/active у dropdown

.dropdown-item.active,
.dropdown-item:focus,
.dropdown-item:active {
  background-color: black !important;
  color: white !important;
}

Тень

Можно использовать готовые классы Bootstrap: shadow-sm, shadow, shadow-lg. Если нужна своя тень:

.my-shadow {
  box-shadow: 0px 5px 10px 0px rgba(0, 0, 0, 0.5);
}

break-inside: avoid

Используется в masonry/columns, чтобы карточка или картинка не разрывалась между колонками.

.masonry img,
.masonry .card {
  break-inside: avoid;
}

Затемнение картинки при наведении

#photo img {
  transition: .3s;
}

#photo img:hover {
  transform: scale(1.02);
  filter: brightness(70%);
}

line-height: 1.5 в Bootstrap

<p class="lh-base">
  Текст с line-height: 1.5;
</p>
